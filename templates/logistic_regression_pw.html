Algorithm Details

How Logistic Regression works here

Logistic Regression estimates the probability of a locality belonging to the HIGH risk class using the sigmoid function: P(HIGH) = 1 / (1 + e^(-z)) where z is the linear combination of standardized features and their coefficients.

All features were normalized using StandardScaler before training, ensuring coefficients are comparable in magnitude. The intercept is {{ intercept }}. Solver: {{ solver }}. Max iterations: {{ max_iter }}.
