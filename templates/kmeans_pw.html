Algorithm Details

How K-Means works here

K-Means initializes {{ k }} centroids, assigns each locality to the nearest one by Euclidean distance, recalculates centroids as the mean of assigned localities, and repeats until convergence. Features were normalized using StandardScaler to prevent scale dominance.

Risk labels (HIGH / MEDIUM / LOW) were assigned post-clustering by ranking clusters on mean abuse level — the cluster with the highest average abuse cases became HIGH, the lowest became LOW. n_init={{ n_init }} random initializations were run to find the optimal solution.
